DataCaptureMVP

A production-ready mobile application for capturing field data — photos and videos — with automatic GPS location, real-time watermarking, categorization, and a powerful admin dashboard.

فیلڈ ڈیٹا کیپچر کے لیے ایک مکمل موبائل ایپلیکیشن — تصاویر اور ویڈیوز خودکار GPS لوکیشن، ریئل ٹائم واٹر مارکنگ، زمرہ بندی، اور ایڈمن ڈیش بورڈ کے ساتھ۔

Features / خصوصیات

📸
Media Capture
Take photos and videos with camera or pick from gallery with automatic quality optimization.
کیمرہ یا گیلری سے فوٹو اور ویڈیو لیں، خودکار کوالٹی کے ساتھ۔
📍
GPS Location
Automatic high-precision GPS coordinates capture at the time of media recording.
میڈیا ریکارڈنگ کے وقت خودکار GPS مقام حاصل ہوتا ہے۔
🔖
Watermarking
GPS coordinates and timestamp are overlaid on photos and videos automatically.
فوٹو اور ویڈیو پر GPS اور وقت خودکار لکھا جاتا ہے۔
🏷️
Categorization
Organize captures by Category (Incident, Observation, Maintenance) and Severity (Low, Medium, High).
زمرہ اور شدت کے مطابق ترتیب دیں۔
📝
Tags & Notes
Add custom tags (Safety, Environment, Equipment, Personnel) and detailed notes.
اپنی مرضی کے ٹیگز اور تفصیلی نوٹس شامل کریں۔
☁️
Cloud Upload
Media and metadata are securely uploaded to Firebase Storage and Firestore.
میڈیا اور ڈیٹا Firebase پر محفوظ طریقے سے اپلوڈ ہوتا ہے۔
👤
Authentication
Anonymous login for field users, Email/Password authentication for administrators.
عام صارفین بغیر اکاؤنٹ، ایڈمن ای میل/پاسورڈ سے لاگ ان۔
📊
Admin Dashboard
View, filter, and manage all uploads. View locations on Google Maps, download media files.
تمام اپلوڈز دیکھیں، فلٹر کریں، نقشے پر مقام دیکھیں۔
🗺️
Map Integration
View exact capture locations on Google Maps with one tap from admin dashboard.
ایڈمن ڈیش بورڈ سے ایک ٹیپ پر Google Maps میں مقام دیکھیں۔
🛠️

Editing Guide / تبدیلی کا طریقہ

🔧 Firebase Configuration تبدیل کرنا
File: lib/main.dart (Lines 23-30)
اپنی Firebase Console سے API Key, Project ID, App ID وغیرہ حاصل کر کے اس فائل میں ڈالیں۔
اینڈرائیڈ کے لیے android/app/google-services.json بھی بدلیں۔
📸 Image Quality / ویڈیو Duration تبدیل کرنا
File: lib/services/media_service.dart
imageQuality: 80 → 100 (بہتر کوالٹی)
maxDuration: Duration(seconds: 30) → 60 (لمبی ویڈیو)
📍 GPS Accuracy تبدیل کرنا
File: lib/services/location_service.dart
LocationAccuracy.high → LocationAccuracy.best
ٹائم آؤٹ شامل کریں: timeLimit: Duration(seconds: 15)
🔖 Watermark Text / Position تبدیل کرنا
File: lib/services/watermark_service.dart
واٹر مارک ٹیکسٹ، رنگ، پوزیشن، اور شفافیت تبدیل کر سکتے ہیں۔
تصاویر: img.drawString() — ویڈیوز: composeVideo(anchor:)
🏷️ Categories / Tags تبدیل کرنا
File: lib/screens/capture_screen.dart (Lines 30-32)
_categories لسٹ میں نئے زمرے شامل کریں
_availableTags لسٹ میں نئے ٹیگز شامل کریں
📊 نئی فیلڈ ڈیٹا ماڈل میں شامل کرنا
File: lib/models/upload_model.dart
1. ماڈل میں نئی فیلڈ شامل کریں
2. toMap() اور fromMap() میں بھی شامل کریں
3. Capture اور Preview سکرینز میں UI شامل کریں
🧭 نئی سکرین/راؤٹ شامل کرنا
File: lib/main.dart (Routes section)
نئی GoRoute() شامل کریں
نیویگیشن: context.push('/your-path')
